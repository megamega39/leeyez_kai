# Leeyez Kai

画像・動画・書庫ビューア for Windows

Unicode対応の [Leeyes](http://www3.toez.net/leeyes/) 後継を目指すネイティブビューアです。

## 特徴

- **書庫の直接閲覧** — ZIP, RAR, 7z, CBZ, CBR 等を展開不要で閲覧
- **見開き表示** — 左綴じ/右綴じ対応、自動見開き判定
- **動画・音声再生** — MP4, MKV, MP3 等に対応
- **本棚・お気に入り** — よく見るフォルダや書庫を整理
- **高速プリフェッチ** — 次の画像を先読みしてスムーズに閲覧
- **多言語対応** — 日本語, English, 中文, 한국어 他12言語
- **ショートカットカスタマイズ** — キーバインドを自由に変更可能

## 対応形式

| 種類 | 形式 |
|------|------|
| 画像 | JPG, PNG, BMP, GIF, WebP, AVIF, TIFF, ICO |
| 動画 | MP4, MKV, AVI, MOV, WMV, WebM, FLV |
| 音声 | MP3, WAV, OGG, FLAC, M4A, AAC |
| 書庫 | ZIP, CBZ, RAR, CBR, 7z, TAR, GZ, BZ2, XZ, ISO, LZH |

## インストール

1. [Releases](https://github.com/megamega39/leeyez_kai/releases) から最新版をダウンロード
2. 任意の場所に配置して実行（インストール不要）

### 動作要件

- Windows 10/11
- .NET 8.0 Runtime ([ダウンロード](https://dotnet.microsoft.com/download/dotnet/8.0))

## ビルド

```bash
dotnet build
dotnet run
```
